# 快速上手 / Quick Start

## 第一步：配置 API Key

1. 打开 OpenCowork
2. 点击右上角 **设置** 图标
3. 选择 **AI 提供商** 标签
4. 选择你的提供商（如 Anthropic）并填入 API Key

<Callout type="tip">
  如果你没有 API Key，可以使用 Ollama 在本地运行模型，无需 API Key。参见 [Ollama 配置](/docs/providers/ollama)。
</Callout>

## 第二步：创建会话

1. 点击左侧边栏的 **新建会话** 按钮
2. 选择模型（如 `claude-sonnet-4-6`）
3. 在输入框中输入你的第一条消息

## 第三步：使用工具

OpenCowork 的 Agent 可以自动调用工具完成任务。例如：

```
帮我读取 ~/Documents/report.txt 并总结内容
```

Agent 会自动调用 `Read` 工具读取文件，然后生成摘要。

## 第四步：启用插件（可选）

如果你想让 OpenCowork 自动回复飞书/钉钉消息：

1. 进入 **设置 → 插件**
2. 选择对应平台并填入配置
3. 开启插件开关

详见 [插件系统](/docs/plugins/overview)。
