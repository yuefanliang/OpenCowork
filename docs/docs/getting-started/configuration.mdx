# 基础配置 / Configuration

## 设置面板 / Settings Panel

通过右上角设置图标打开设置面板，包含以下配置项：

### API Key 配置

每个 AI 提供商需要单独配置 API Key：

| 提供商 | 配置项 |
|--------|--------|
| Anthropic | `ANTHROPIC_API_KEY` |
| OpenAI | `OPENAI_API_KEY` + 可选 Base URL |
| DeepSeek | API Key + Base URL |
| Ollama | Base URL（默认 `http://localhost:11434`） |

### 语言设置

支持中文（简体）和英文界面切换，设置后立即生效。

### 主题设置

支持亮色、暗色、跟随系统三种主题模式。

### 工作目录

设置 Agent 的默认工作目录，影响文件读写和 Shell 命令的执行路径。

### 模型默认值

可以为每个会话设置默认模型，也可以在会话中临时切换。

## 数据持久化 / Data Persistence

所有配置通过 Zustand persist 中间件持久化到本地存储（`localStorage`），数据库内容存储在 `~/.open-cowork/data.db`。
