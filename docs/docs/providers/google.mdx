# Google Gemini

Google Gemini 提供超长上下文窗口和强大的多模态能力。

## 获取 API Key

1. 访问 [aistudio.google.com](https://aistudio.google.com/)
2. 点击 **Get API Key**
3. 创建新的 API Key

## 在 OpenCowork 中配置

进入 **设置 → AI 提供商**，添加新提供商：

| 配置项 | 值 |
|--------|-----|
| 协议 | `openai-chat` |
| Base URL | `https://generativelanguage.googleapis.com/v1beta/openai` |
| API Key | 你的 Gemini API Key |

## 支持的模型 / Supported Models

| 模型 | 上下文 | 特性 |
|------|--------|------|
| `gemini-2.0-flash` | 1M | 快速、多模态 |
| `gemini-2.5-pro` | 1M | 最强能力 |
| `gemini-2.0-flash-thinking` | 1M | 推理模式 |

## 视觉能力 / Vision

Gemini 支持图片、视频、音频输入，可以在聊天中直接上传多媒体文件。
