# DeepSeek

DeepSeek 提供高性价比的大语言模型，包括强大的推理模型 DeepSeek-R1。

## 获取 API Key

1. 访问 [platform.deepseek.com](https://platform.deepseek.com/)
2. 注册账号并充值
3. 在 **API Keys** 页面创建密钥

## 在 OpenCowork 中配置

进入 **设置 → AI 提供商**，添加新提供商：

| 配置项 | 值 |
|--------|-----|
| 协议 | `openai-chat` |
| Base URL | `https://api.deepseek.com/v1` |
| API Key | 你的 DeepSeek API Key |

## 支持的模型 / Supported Models

| 模型 | 特性 |
|------|------|
| `deepseek-chat` | 通用对话，高性价比 |
| `deepseek-reasoner` | 推理模型（R1），复杂问题求解 |

## 推理模式 / Reasoning Mode

`deepseek-reasoner` 支持推理模式，会在回答前输出思考过程（`<think>` 标签）。OpenCowork 会自动解析并在 UI 中展示思考内容。
