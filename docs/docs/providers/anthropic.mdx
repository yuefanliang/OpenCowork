# Anthropic / Claude

Anthropic 是 OpenCowork 的首选提供商，支持 Claude 全系列模型。

## 获取 API Key

1. 访问 [console.anthropic.com](https://console.anthropic.com/)
2. 注册/登录账号
3. 进入 **API Keys** 页面
4. 点击 **Create Key**

## 在 OpenCowork 中配置

进入 **设置 → AI 提供商 → Anthropic**，填入 API Key。

## 支持的模型 / Supported Models

| 模型 | 上下文 | 特性 |
|------|--------|------|
| `claude-opus-4-6` | 200K | 最强能力，扩展思考 |
| `claude-sonnet-4-6` | 200K | 性能与速度平衡 |
| `claude-haiku-4-5` | 200K | 快速、低成本 |

## 扩展思考 / Extended Thinking

Claude 支持扩展思考模式，让模型在回答前进行深度推理：

在会话设置中开启 **Thinking** 模式，Agent 循环会在响应中包含 `thinking_delta` 事件。

## 视觉能力 / Vision

所有 Claude 模型支持图片输入，可以直接在聊天中粘贴或上传图片。
