# 钉钉集成 / DingTalk Integration

OpenCowork 通过 WebSocket 连接钉钉开放平台，实现消息自动回复。

## 前置要求 / Prerequisites

1. 在[钉钉开放平台](https://open.dingtalk.com/)创建企业内部应用
2. 开启机器人能力
3. 获取 AppKey 和 AppSecret

## 配置步骤 / Setup

### 1. 创建钉钉应用

1. 访问钉钉开放平台 → 应用开发
2. 创建企业内部应用
3. 在「消息推送」中开启机器人

### 2. 配置机器人

在机器人配置页面：

- 设置机器人名称和头像
- 开启「单聊」和/或「群聊」消息接收

### 3. 在 OpenCowork 中配置

进入 **设置 → 插件 → 钉钉**，填入：

| 配置项 | 说明 |
|--------|------|
| AppKey | 钉钉应用 AppKey |
| AppSecret | 钉钉应用 AppSecret |

### 4. 开启插件

点击开关启用钉钉插件。

## 消息类型支持 / Message Types

| 类型 | 接收 | 发送 |
|------|------|------|
| 文本 | ✅ | ✅ |
| Markdown | ❌ | ✅ |
| 图片 | ✅ | ✅ |
