# Discord 集成 / Discord Integration

OpenCowork 通过 Discord Bot API 实现服务器消息自动回复。

## 创建 Bot / Create Bot

1. 访问 [Discord Developer Portal](https://discord.com/developers/applications)
2. 点击 **New Application** 创建应用
3. 进入 **Bot** 标签，点击 **Add Bot**
4. 复制 **Token**

## 配置权限 / Permissions

在 **OAuth2 → URL Generator** 中选择：

- Scopes: `bot`
- Bot Permissions: `Send Messages`, `Read Message History`, `View Channels`

使用生成的 URL 将 Bot 邀请到你的服务器。

## 在 OpenCowork 中配置 / Configure

进入 **设置 → 插件 → Discord**，填入：

| 配置项 | 说明 |
|--------|------|
| Bot Token | Discord Bot Token |

## 触发方式 / Trigger

在频道中 @ 机器人触发回复：

```
@YourBot 帮我分析这段代码
```
