# 企业微信集成 / WeCom Integration

OpenCowork 通过企业微信应用 API 实现消息自动回复。

## 前置要求 / Prerequisites

- 企业微信管理员权限
- 在[企业微信管理后台](https://work.weixin.qq.com/wework_admin/)创建自建应用

## 配置步骤 / Setup

### 1. 创建自建应用

1. 登录企业微信管理后台
2. 进入 **应用管理 → 自建 → 创建应用**
3. 设置应用名称、Logo 和可见范围

### 2. 获取凭证

在应用详情页获取：

- `企业 ID (CorpID)`：在「我的企业」页面
- `AgentID`：应用详情页
- `Secret`：应用详情页

### 3. 配置消息接收

在应用的「接收消息」设置中：

1. 开启消息接收
2. 设置 URL（OpenCowork 提供本地地址）
3. 填写 Token 和 EncodingAESKey

### 4. 在 OpenCowork 中配置

进入 **设置 → 插件 → 企业微信**，填入：

| 配置项 | 说明 |
|--------|------|
| CorpID | 企业 ID |
| AgentID | 应用 AgentID |
| Secret | 应用 Secret |
| Token | 消息接收 Token |
| EncodingAESKey | 消息加密密钥 |
