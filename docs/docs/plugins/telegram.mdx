# Telegram 集成 / Telegram Integration

OpenCowork 通过 Telegram Bot API 实现消息自动回复。

## 创建 Bot / Create Bot

1. 在 Telegram 中找到 [@BotFather](https://t.me/BotFather)
2. 发送 `/newbot` 创建新机器人
3. 按提示设置机器人名称和用户名
4. 获取 **Bot Token**（格式：`123456789:ABCdefGHI...`）

## 在 OpenCowork 中配置 / Configure

进入 **设置 → 插件 → Telegram**，填入：

| 配置项 | 说明 |
|--------|------|
| Bot Token | BotFather 提供的 Token |

点击开关启用插件。

## 使用方式 / Usage

- **私聊**：直接向机器人发送消息
- **群组**：将机器人添加到群组，@ 机器人触发回复

## 注意事项 / Notes

<Callout type="info">
  Telegram 使用长轮询（Long Polling）接收消息，无需公网 IP 或 webhook 配置。
</Callout>
