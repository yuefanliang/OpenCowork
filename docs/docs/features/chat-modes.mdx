# 聊天模式 / Chat Modes

OpenCowork 提供两种聊天模式，通过顶部工具栏切换。

## Cowork 模式

通用 AI 助手模式，适合日常对话、文档处理、信息查询等任务。

系统提示侧重于协作助手角色，工具调用更保守，适合非技术用户。

## Code 模式

代码专家模式，适合软件开发任务：

- 系统提示包含代码最佳实践指导
- 默认启用文件系统工具（Read/Write/Edit）
- 启用 Shell 工具（Bash）
- 启用搜索工具（Glob/Grep）

## 工作目录 / Working Directory

在 Code 模式下，可以设置工作目录，Agent 的文件操作和 Shell 命令都在该目录下执行：

```
设置 → 工作目录 → 选择文件夹
```

工作目录信息会注入到系统提示中，让 Agent 了解当前项目上下文。

## 模式切换 / Switching Modes

模式切换通过顶部工具栏的切换按钮完成。切换后，系统提示会立即更新，但不会影响当前会话的历史消息。

<Callout type="info">
  手动切换模式后，OpenCowork 会记住你的选择，不会被自动同步覆盖。
</Callout>
