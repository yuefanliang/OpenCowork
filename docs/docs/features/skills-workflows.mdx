# 技能与工作流 / Skills & Workflows

技能（Skills）是预定义的提示模板，工作流（Workflows）是可复用的 Agent 执行序列。

## 技能 / Skills

技能通过 `Skill` 工具调用：

```json
{
  "skill": "commit",
  "args": "-m 'Fix authentication bug'"
}
```

### 内置技能

| 技能名 | 描述 |
|--------|------|
| `commit` | 生成规范的 Git commit 消息 |
| `review-pr` | 审查 Pull Request |
| `web-scraper` | 抓取网页内容 |

### 自定义技能

技能定义存储在 `~/.open-cowork/workflows/` 目录下的 `.md` 文件中：

```markdown
---
name: my-skill
description: 我的自定义技能
---

执行以下操作：
1. 读取 {args} 指定的文件
2. 分析内容
3. 生成报告
```

## 工作流 / Workflows

工作流是更复杂的自动化序列，支持条件分支和循环。工作流文件同样存储在 `~/.open-cowork/workflows/` 目录。

## 斜杠命令 / Slash Commands

在聊天输入框中输入 `/` 可以快速调用技能：

```
/commit
/review-pr 123
```
