# MCP 服务器 / MCP Servers

OpenCowork 支持 Model Context Protocol (MCP)，可以连接外部 MCP 服务器扩展 Agent 能力。

## 什么是 MCP / What is MCP

MCP 是 Anthropic 提出的开放协议，允许 AI 应用连接外部工具和数据源。通过 MCP，你可以：

- 连接数据库（PostgreSQL、MySQL、SQLite）
- 访问外部 API（GitHub、Slack、Notion）
- 使用专业工具（代码执行、浏览器控制）

## 添加 MCP 服务器 / Add MCP Server

1. 进入 **设置 → MCP 服务器**
2. 点击 **添加服务器**
3. 填写服务器配置：

```json
{
  "name": "github",
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-github"],
  "env": {
    "GITHUB_PERSONAL_ACCESS_TOKEN": "your-token"
  }
}
```

## 支持的传输方式 / Transport

| 传输方式 | 描述 |
|---------|------|
| `stdio` | 通过标准输入输出通信（本地进程） |
| `sse` | Server-Sent Events（HTTP 服务器） |

## 工具发现 / Tool Discovery

连接 MCP 服务器后，OpenCowork 会自动发现并注册服务器提供的工具，Agent 可以直接使用这些工具。

## 管理服务器 / Manage Servers

通过 `mcp-store` 管理 MCP 服务器状态：

- 查看连接状态（connected / disconnected / error）
- 启用/禁用服务器
- 查看服务器提供的工具列表
